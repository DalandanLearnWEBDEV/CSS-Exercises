# CSS-Exercises
**CSS**, which stands for _Cascading Style Sheet_, is a crucial component of modern web development  
It's a markup language used to apply styles for HTML. In Simple terms, if **HTML** is the structure  
web page, **CSS** is what makes it look good.<br>
## Exercises  
- Cafe Menu  
- Business Card  
- To Do List  
- Blog Post Card  
- Event Flyer Page
